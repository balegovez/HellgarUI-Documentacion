Alerts

Mensajes contextuales para feedback rápido al usuario.

En esta página
  • Alerta base
  • Variantes

Alerta básica

La clase base es .alert, pero sola no muestra colores. Le tenes que sumar una variante semantica: .alert-success, .alert-danger, .alert-warning o .alert-info.

Vista previa
Operación completada con éxito. 
<div class="alert alert-success">
    Operación completada con éxito.
</div>

Las 4 variantes

Cada variante esta pensada para un tipo de mensaje distinto. Verde para confirmar, rojo para errores, amarillo para advertencias y azul para info neutral.

Todas juntas
Éxito: los cambios se guardaron.
Error: no se pudo conectar con el servidor.
Atención: tu sesión expira en 2 minutos.
Info: hay una nueva versión disponible. 
<div class="alert alert-success">Éxito</div>
<div class="alert alert-danger">Error</div>
<div class="alert alert-warning">Atención</div>
<div class="alert alert-info">Info</div>

Alerta con título y descripción

Adentro de la alerta podes usar headings y parrafos. Combinalos con utilidades de tipografia y spacing para darle estructura.

Mas contenido adentro

Cuidado con esto

Estas a punto de borrar tu cuenta. Esta acción no se puede deshacer ni llorando. 

<div class="alert alert-warning">
    <h4 class="fs-5 fw-bold mb-2">Cuidado con esto</h4>
    <p class="m-0">
        Estas a punto de borrar tu cuenta.
    </p>
</div>

Alertas con enlaces

Si necesitas agregar un link adentro de la alerta, le podes meter un <a> con utilidades de tipografía como fw-bold para que se note bien.

Con call to action
Hay actualizaciones pendientes. Ver detalles 
<div class="alert alert-info">
    Hay actualizaciones pendientes.
    <a href="#" class="fw-bold">Ver detalles</a>
</div>

Combinada con otros componentes

Dentro de un card-body, una alerta queda muy bien para mostrar un estado sobre un formulario o una accion. Tambien podes ponerle un boton al lado.

Alerta con acción
Conexión perdida. No se pudo guardar el formulario. 
<div class="alert alert-danger">
    <div class="d-flex justify-content-between align-items-center flex-wrap gap-2">
        <span><strong>Conexión perdida.</strong> No se pudo guardar.</span>
        <button class="btn btn-danger btn-sm">Reintentar</button>
    </div>
</div>
