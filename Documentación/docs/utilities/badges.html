Badges

Etiquetas pequeñas para estados, contadores o categorías.

En esta página
  • Badge básico
  • Variantes
  • Forma cápsula
  • En la práctica

Badge básico

La clase base es .badge y por sí sola ya tiene un padding, un radio y un texto en negrita. Sumale una variante de color para ver algo.

Vista previa
primary 
<span class="badge badge-primary">primary</span>

Variantes de color

Tres variantes de la marca: .badge-primary, .badge-secondary y .badge-accent. Las dos primeras llevan texto blanco; la accent (naranja brasa) usa texto oscuro para mantener contraste.

Vista previa
primary secondary accent 
<span class="badge badge-primary">primary</span>
<span class="badge badge-secondary">secondary</span>
<span class="badge badge-accent">accent</span>

Forma cápsula

Sumá .badge-pill al lado de cualquier variante para redondear los bordes por completo y obtener una etiqueta tipo píldora.

Vista previa
primary secondary accent 
<span class="badge badge-primary badge-pill">v1.0</span>
