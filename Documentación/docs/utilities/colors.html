Paleta principal

Tres colores de marca, inspirados en Houndoom (siniestro/fuego): un rojo intenso, un negro carbón y un naranja brasa. Más dos neutrales (claro y oscuro) que se usan como fondo y texto base.

Clases de texto

Para pintar el color de un texto sumá la clase .text-{nombre}. Funciona con cualquier color de la paleta y con los semánticos.

Clases de fondo

Para fondos usá .bg-{nombre}. Se aplica a cualquier elemento (sections, cards, divs). Notá que el texto se ajusta automáticamente, ver la sección de auto-contraste.

Auto-contraste

Los colores con fondo oscuro (primary, secondary, success, danger, info, dark) ya aplican color de texto claro automáticamente. No tenés que sumar .text-white. Si querés forzar otro color, simplemente sobreescribilo con la clase explícita.

Ejemplo
Sin clase de texto: blanco automático.
Con .text-warning: el sistema respeta tu override.
Fondos claros: texto oscuro automático. 
<div class="bg-primary">Texto blanco automático</div>
<div class="bg-primary text-warning">Override manual</div>
