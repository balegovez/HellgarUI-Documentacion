Footers (Pie de página)

Estructuras flexibles para el final de tus páginas. Soportan múltiples columnas, enlaces de interés y variantes de color.

En esta página
  • Footer Base (Dark)
  • Footer Light
  • Footer Secondary
  • Footer Simple

Footer Base (Dark)

Por defecto, el componente .footer utiliza el color $dark. Utiliza la estructura interna .footer-row y .footer-col para distribuir el contenido de forma responsiva.

Footer Light

Agrega la clase modificadora .footer-light junto a .footer para invertir los colores y utilizar la variable $light. Ideal para diseños de fondo blanco donde quieres mantener la limpieza visual.

Footer Minimalista

No es obligatorio usar toda la cuadrícula. Puedes omitir .footer-row y utilizar solo la clase base para crear un footer pequeño y centrado. Para alinear el texto, puedes apoyarte en tus clases utilitarias[cite: 21].
