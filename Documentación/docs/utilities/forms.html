Formulario básico

El patrón es siempre el mismo: un .form-group que envuelve un .form-label y un input con .form-control. El group ya viene con un margen inferior asi que no necesitas sumar mb-*. Puedes agregar .form-text para textos de ayuda.

Vista previa
Nunca compartiremos tu correo con nadie. 
<div class="form-group">
    <label class="form-label" for="email">Email</label>
    <input type="email" id="email" class="form-control" placeholder="tucorreo@ejemplo.com">
    <small class="form-text">Nunca compartiremos tu correo con nadie.</small>
</div>

Tipos de input

La clase .form-control sirve para casi cualquier tipo de input HTML5: text, email, password, number, date y compañia.

Varios tipos
<input type="text" class="form-control" placeholder="hellgar_user">
<input type="password" class="form-control" placeholder="********">
<input type="number" class="form-control" placeholder="18">
<input type="date" class="form-control">

Variantes de estilo

Cuatro estilos de input además del clásico, para adaptarse al diseño de tu interfaz.

Clases de variantes
<input class="form-control form-control-outline">
<input class="form-control form-control-flush">
<input class="form-control form-control-filled">
<input class="form-control form-control-pill">

Textarea y select

Los <textarea> y <select> tambien usan .form-control. Funcionan igual y respetan el mismo estilo de borde y focus.

Textarea + select
<textarea class="form-control" rows="4" placeholder="..."></textarea>

<select class="form-control">
    <option>El Salvador</option>
    <option>Argentina</option>
</select>

Input group

Envolvé tu input en un .input-group y añade addons antes o después usando .input-group-text.

Addons
@
$ USD 
<div class="input-group">
    <span class="input-group-text">$</span>
    <input type="number" class="form-control" placeholder="0.00">
    <span class="input-group-text">USD</span>
</div>

Floating label

Usa .form-floating para etiquetas animadas. Requiere que el input tenga el atributo placeholder (incluso si está vacío) y que la clase .form-label venga después del input en el DOM.

Etiquetas flotantes
<div class="form-floating">
    <input type="email" class="form-control" id="floatingInput" placeholder="name@example.com">
    <label class="form-label" for="floatingInput">Correo electrónico</label>
</div>

Checkboxes y radios

Para checks y radios usá un <label> con la clase .form-check. Adentro mete el input y el texto.

Checks apilados

Estados de validación

Sumale .is-valid al input cuando esté OK, o .is-invalid cuando tenga error. El borde y el halo de focus cambian del color correspondiente.

Estados admitidos
Se ve bien.
Falta el @ y el dominio. 
<input class="form-control is-valid">
<input class="form-control is-invalid">

Formulario completo

Combinando todo lo de arriba mas el sistema de grid podes armar formularios de varias columnas en desktop y una sola columna en mobile, sin pelear nada.

Form de contacto
<div class="row">
    <div class="col-12 col-md-6">
        <div class="form-group">
            <label class="form-label" for="nombre">Nombre</label>
            <input type="text" id="nombre" class="form-control">
        </div>
    </div>
    <div class="col-12 col-md-6">
        <div class="form-group">
            <label class="form-label" for="mail">Email</label>
            <input type="email" id="mail" class="form-control">
        </div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary">Enviar mensaje</button>
    </div>
</div>
