¿Qué es HellgarUI?

HellgarUI es un framework CSS construido en SASS, con una arquitectura modular dividida en base, layout, components y utilities. Si vienes de Bootstrap te vas a sentir cómodo: 12 columnas, breakpoints sm/md/lg/xl/xxl, utilidades de spacing y un sistema de colores semántico.

La diferencia es que es chiquito, sin JavaScript y con una identidad visual propia (rojo, negro y naranja).

Instalación

Solo enlaza el main.css compilado en el head de tu HTML y listo. No hay paquetes, no hay build steps obligatorios para el usuario final.

Paso 1 · Enlazar el CSS
<link rel="stylesheet" href="main.css">
Paso 2 · Primer "hola mundo"
<div class="container py-5">
    <h1 class="fs-1 text-primary">¡Hola Mundo!</h1>
    <button class="btn btn-primary">Botón de prueba</button>
</div>

Filosofía

📱

Mobile first

Todas las clases base aplican primero a móvil y se expanden hacia arriba con los breakpoints.

🧩

Modular

Carga solo lo que vayas a usar editando el main.scss. No infles el CSS sin razón.

😎

Personalidad

Los colores no son aburridos. Tampoco la documentación (eso esperamos).
